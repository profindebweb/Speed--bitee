# 🍔 Speed Bite

نظام طلبات المطعم السريع بالدارجة المغربية.

## المميزات
- ✅ واجهة زبون أنيقة مع صور حقيقية
- ✅ لوحة مطبخ حية مع Realtime
- ✅ تصنيفات الطعام (برݣر، تاكوس، بيتزا...)
- ✅ سلة تسوق مع ملاحظات
- ✅ إشعارات فورية للطلبيات الجديدة
- ✅ إحصائيات يومية
- ✅ تصميم داكن احترافي

## التقنيات
- Next.js 15 + React 19
- TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- Supabase (Realtime)
- الدارجة المغربية 🇲🇦
